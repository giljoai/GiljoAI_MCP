DATABASE SCHEMA
================================================================================

Database: giljo_mcp
  Owner: giljo_owner

Roles:
  giljo_owner  - Database owner, runs migrations, full privileges
  giljo_user   - Application runtime, read/write data only

Permissions:
  giljo_owner  - USAGE, CREATE on schema public
  giljo_user   - USAGE on schema public
               - SELECT, INSERT, UPDATE, DELETE on tables
               - USAGE, SELECT on sequences

================================================================================
INSTALLATION FLOW
================================================================================

1. Check PostgreSQL Connection
   └─→ Fail: Return installation guide

2. Detect PostgreSQL Version
   ├─→ < 14: Block installation
   ├─→ 14-17: Warn, continue
   └─→ 18: Recommended, continue

3. Try Direct Database Creation
   ├─→ Success:
   │   ├─→ Create roles
   │   ├─→ Create database
   │   ├─→ Setup permissions
   │   └─→ Save credentials
   │
   └─→ Fail (permissions):
       ├─→ Generate fallback script
       ├─→ Display instructions
       ├─→ Wait for user execution
       └─→ Verify database creation

4. Run Alembic Migrations
   └─→ Initialize schema

5. Return Success
