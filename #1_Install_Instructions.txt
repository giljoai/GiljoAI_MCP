Thank you for downloading GiljoAI MCP!

Quick Install (recommended)
───────────────────────────
  Windows (PowerShell):
    irm https://raw.githubusercontent.com/giljoai/GiljoAI_MCP/master/scripts/install.ps1 | iex

  Linux / macOS:
    curl -fsSL https://raw.githubusercontent.com/giljoai/GiljoAI_MCP/master/scripts/install.sh | bash

  The installer handles everything: checks for Python and PostgreSQL,
  downloads the latest release, sets up the database, builds the frontend,
  and walks you through configuration.

  When done, start the server:
    python startup.py

Manual Install
──────────────
  Prerequisites:
    - Python 3.10+ (3.12+ recommended)
    - PostgreSQL 14+ (18 recommended)
    - Node.js 20 LTS

  Steps:
    git clone https://github.com/giljoai/GiljoAI_MCP.git
    cd GiljoAI_MCP
    python install.py
    python startup.py

For detailed instructions, see docs/INSTALLATION_GUIDE.md
