Smooth Borders (MANDATORY)

NEVER use CSS border on rounded elements. Use box-shadow: inset instead — it renders anti-aliased curves on all zoom levels and Windows display scaling, avoiding the pixelated staircase effect of thin CSS borders. Global utility classes are defined in main.scss.

content
.smooth-border
1px theme inset 0 0 0 1px #315074
content
.smooth-border-2
2px theme inset 0 0 0 2px #315074
content
.smooth-border-accent
2px brand yellow inset 0 0 0 2px #ffc300
DO NOT USE
CSS border
pixelated on zoom border: 1px solid #315074

Custom color override: style="--smooth-border-color: #hex"
For chips/pills: use box-shadow: inset 0 0 0 2px var(--color-accent-primary) in scoped CSS with transparent background.
