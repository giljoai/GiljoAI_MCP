3. PCI Bus Architecture: Validated Performance Foundation
Modern Platform Capabilities:
o X670E Motherboards: 2x PCIe 5.0 x16 slots (CPU direct) supporting x16/x8 configuration
o PCIe 5.0 x8 bandwidth: ~32 GB/s per slot - exceeds RTX Ada card requirements
o Real bottlenecks: GPU memory bandwidth, storage I/O, not PCI bus transfers
AI Workload Reality:
o LLM inference: Model loads once, 95% internal GPU computation
o Document processing: Small transfers (1-10MB images), GPU-internal processing
o Concurrent operation: Both GPUs rarely saturate PCI bandwidth simultaneously
o Conclusion: PCI bus bandwidth provides 2-4x headroom over typical AI workload requirements.

    def detect_contact_format(self, csv_path):
        # Intelligent detection of contact CSVs
        headers = self.read_csv_headers(csv_path)
        contact_indicators = ['email', 'phone', 'first_name', 'last_name', 'company']
        matches = sum(1 for ind in contact_indicators if ind in headers.lower())

API Rate Limiting and Resource Management:
pythonclass APIRateLimiter:
    def __init__(self):
        self.limits = {
            'document_processing': {'per_minute': 10, 'concurrent': 3},
            'knowledge_graph_queries': {'per_minute': 100, 'complexity_budget': 1000},
            'entity_extraction': {'per_minute': 50}
        }
    
    def check_rate_limit(self, endpoint, user_id):
        # Per-user rate limiting to ensure fair resource usage
        current_usage = self.get_current_usage(endpoint, user_id)
        limit = self.limits[endpoint]
        
        if current_usage.exceeds(limit):
            return False, f"Rate limit exceeded. Try again in {current_usage.reset_time}s"
        
        return True, None

7. API Authentication and Security
JWT-Based Authentication:
pythonclass APIAuthenticationManager:
    def __init__(self):
        self.algorithm = 'RS256'  # Asymmetric for better security
        self.token_expiry = 3600  # 1 hour
        
    def generate_api_token(self, user_id, scopes):
        payload = {
            'user_id': user_id,
            'scopes': scopes,
            'exp': datetime.utcnow() + timedelta(seconds=self.token_expiry),
            'iat': datetime.utcnow()
        }
        
        return jwt.encode(payload, self.private_key, algorithm=self.algorithm)
    
    def validate_token(self, token):
        try:
            payload = jwt.decode(token, self.public_key, algorithms=[self.algorithm])
            return True, payload
        except jwt.ExpiredSignatureError:
            return False, "Token expired"
        except jwt.InvalidTokenError:
            return False, "Invalid token"
