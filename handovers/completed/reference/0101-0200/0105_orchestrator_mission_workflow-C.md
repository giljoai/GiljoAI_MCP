## Executive Summary

This handover documents the **existing orchestrator mission workflow** and adds a **NEW FEATURE**: Claude Code subagent toggle for the Implementation tab. The system already provides mission visibility with a simple cancel/proceed workflow. The enhancement allows users to indicate when using Claude Code's native subagent spawning, which grays out individual agent prompt buttons (only orchestrator button remains active).

**Current Workflow**:
1. User activates project → Orchestrator job created (status: `waiting`)
2. Orchestrator analyzes vision/context → Writes mission to `Project.mission`
3. Orchestrator selects agents → Creates agent jobs (status: `waiting`)
4. User reviews mission + agents in Launch tab
5. User clicks "Implement Now" → Switches to Implementation tab
6. User pastes prompts (individual terminals OR Claude Code single terminal)
7. Agents execute using MCP tools

**NEW FEATURE**: Claude Code subagent toggle on Implementation tab
- Toggle indicates "Using Claude Code subagents"
- When ON: All non-orchestrator prompt buttons grayed out
- Only orchestrator button active (Claude spawns subagents automatically)
- Templates enforce MCP communication protocols (see Handover 0106)

## Technical Requirements

### 1. Frontend Enhancement (NEW FEATURE)

**Component**: `frontend/src/components/projects/ImplementationTab.vue`

**Add Toggle Control**:
```vue
<template>
  <v-card>
    <!-- NEW: Claude Code Toggle -->
    <v-card-title>
      <v-switch
        v-model="usingClaudeCodeSubagents"
        label="Using Claude Code subagents"
        color="primary"
        density="compact"
        :hint="toggleHint"
        persistent-hint
      />
    </v-card-title>

    <!-- Agent Grid -->
    <v-row>
      <v-col v-for="agent in agents" :key="agent.id" cols="12" md="6">
        <AgentCardEnhanced
          :agent="agent"
          :prompt-button-disabled="shouldDisablePromptButton(agent)"
        />
      </v-col>
    </v-row>
  </v-card>
</template>

<script>
export default {
  data() {
    return {
      usingClaudeCodeSubagents: false
    }
  },
  computed: {
    toggleHint() {
      return this.usingClaudeCodeSubagents
        ? 'Only orchestrator prompt active - Claude will spawn subagents automatically'
        : 'All agent prompts active - use individual terminals'
    }
  },
  methods: {
    shouldDisablePromptButton(agent) {
      // Disable all non-orchestrator buttons when toggle is ON
      if (this.usingClaudeCodeSubagents) {
        return agent.agent_type !== 'orchestrator'
      }
      return false
    }
  }
}
</script>
```
