## PHASE 1: EXECUTION

### Task 1: ERA001 - Commented-Out Code (~277 instances)

**Strategy:** Delete all commented-out code blocks. We have git history.

```bash
# Find all ERA001 violations
ruff check src/ api/ --select ERA001 --output-format=json > era001_issues.json
```

For each file:
1. Review the commented block
2. If it's dead code → DELETE
3. If it's a TODO/note → Convert to proper comment or issue

**Decision tree:**
- `# old_function()` → DELETE
- `# TODO: implement` → Keep as TODO or create issue
- `# HACK: workaround for X` → Keep with explanation
- `# return old_value` → DELETE

### Task 2: BLE001 - Blind Except (~233 instances)

**Strategy:** Replace bare `except:` with specific exceptions.

```python
# BEFORE (bad)
try:
    something()
except:
    pass

# AFTER (good)
try:
    something()
except Exception as e:
    logger.warning(f"Failed: {e}")
```

**Common patterns:**
- `except:` → `except Exception:`
- `except:` with logging → Keep logging, add exception type
- `except:` with pass → Consider if truly needed or remove try/except

### Task 3: B904 - Raise Without From (~152 instances)

**Strategy:** Add exception chaining.

```python
# BEFORE (bad)
try:
    something()
except ValueError:
    raise CustomError("Failed")

# AFTER (good)
try:
    something()
except ValueError as e:
    raise CustomError("Failed") from e
```

### Task 4: T201 - Print Statements (~37 instances)

**Strategy:** Replace with proper logging or remove.

```python
# BEFORE
print(f"Debug: {value}")

# AFTER
logger.debug(f"Debug: {value}")
```

**Exceptions to keep:**
- CLI scripts that intentionally print to stdout
- Setup/install scripts
- User-facing messages in CLI tools

### Task 5: Other High-Value Fixes

Review and fix:

- `SIM102` collapsible if (18 instances)
- `RUF012` mutable class default (18 instances)
