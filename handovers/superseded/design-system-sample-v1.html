Smooth Borders (MANDATORY)

NEVER use CSS border on rounded elements. Use box-shadow: inset instead — it renders anti-aliased curves on all zoom levels and Windows display scaling, avoiding the pixelated staircase effect of thin CSS borders. Global utility classes are defined in main.scss.

content
.smooth-border
1px theme inset 0 0 0 1px #315074
content
.smooth-border-2
2px theme inset 0 0 0 2px #315074
content
.smooth-border-accent
2px brand yellow inset 0 0 0 2px #ffc300
DO NOT USE
CSS border
pixelated on zoom border: 1px solid #315074

Custom color override: style="--smooth-border-color: #hex"
For chips/pills: use box-shadow: inset 0 0 0 2px var(--color-accent-primary) in scoped CSS with transparent background.

Discrepancies — All Resolved

Previously found mismatches between design-tokens.scss, agent-colors.scss, agentColors.js, theme.js, and main.scss. All resolved in 0769d sprint (2026-03-30).

All 4 discrepancies resolved. Agent colors, status colors, font family, and primary text color are now fully synchronized across design-tokens.scss, agent-colors.scss, agentColors.js, theme.js, and main.scss.

A. Agent Colors — RESOLVED

Analyzer (#e74c3c), Implementor (#3498db), Tester (#ffc300) — all synchronized across all 3 sources.

B. Status Colors — RESOLVED

waiting (#ffd700), working (#ffffff), complete (#67bd6d), failed (#c6298c) — design-tokens.scss and agent-colors.scss now match.

C. Font Family — RESOLVED

Both design-tokens.scss and main.scss now use "Roboto", sans-serif. The "Cairo" reference has been removed.

D. Primary Text Color — RESOLVED

All sources now use #e1e1e1. The #e0e0e0 variant has been removed from design-tokens.scss.

E. Duplicate Radius Systems — RESOLVED (Handover 0277)

Consolidated to 4 canonical tokens in design-tokens.scss: $border-radius-sharp (4px), $border-radius-default (8px), $border-radius-rounded (16px), $border-radius-pill (9999px). All aliases and variables.scss duplicates removed.

F. Duplicate Shadow Systems — RESOLVED (Handover 0277)

Consolidated to 4 semantic tokens in design-tokens.scss: $shadow-subtle, $shadow-default, $shadow-strong, $shadow-dramatic. Old $shadow-sm/md/lg/xl from variables.scss removed.

Card Framing Standard

All rounded cards use box-shadow: inset (smooth-border) — NEVER CSS border. See CLAUDE.md.
