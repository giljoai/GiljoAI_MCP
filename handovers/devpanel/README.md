# Developer Panel – Scope of Work (SoW) Pack

This folder contains a multi‑phase, numbered scope for the temporary Developer Panel surfaced at `/developer`. The panel is a localhost‑only, gated utility to explore architecture, APIs, MCP tools, agents/templates, DB schema, dependency graphs, and configuration surfaces, with search and “PowerPoint‑style” diagram exports. It will be removed prior to shipment or isolated to a dev‑only branch.

Reference sources include project docs under `docs/` and handovers under `handovers/`, reconciled against the live code in `api/` and `src/giljo_mcp/`.

Phases (open each file for full details):

- 1001 – Discovery and Audit
- 1002 – IA and UX Blueprint
- 1003 – Backend Data Services
- 1004 – Frontend Scaffold (/developer)
- 1005 – Diagrams and PPT Exports
- 1006 – Configuration Viewer/Editor
- 1007 – Dependency & Search Explorer
- 1008 – Tech Stack & Deprecations
- 1009 – Security, Gating, Removal
- 1010 – Documentation, QA, Closeout

Deliverables are designed to be shippable per phase. Each phase file includes objectives, scope, deliverables, acceptance criteria, data sources (docs + code), and risks.
